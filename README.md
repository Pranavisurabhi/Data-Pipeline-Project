Project Description

This project demonstrates a data pipeline for preprocessing, transforming, and loading a house price dataset using Python, Pandas, and Scikit-learn.

🎯 Objective

To clean and prepare raw data for machine learning by building an automated data pipeline (ETL process).

⚙️ Steps:
Data Loading
Handling Missing Values
Removing Duplicates
Label Encoding (Categorical Data)
Feature Scaling (Numerical Data)
Saving Processed Data

🛠️ Tools & Technologies:
Python
Pandas
NumPy
Scikit-learn
📂 Files in Repository:
House Price Prediction Dataset.csv 
cleaned_house_data.csv

🚀 Output:
A cleaned and processed dataset ready for machine learning models.

📊 Conclusion:
The data pipeline successfully converts raw data into a structured format suitable for analysis and predictive modeling.
